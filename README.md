gi# imageconverter

It's just a very simple PoC'

Tools used :

1. Spring Boot 1.5.9.RELEASE
2. Spring 4.3.7.RELEASE
3. Thymeleaf
4. Maven
5. Embedded Tomcat 8.5.6
6. iCafe

How to run this repo :

open your terminal then type this command :
`mvn clean spring-boot:run`

open your browser :
`http://localhost:8080`
